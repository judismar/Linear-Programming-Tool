Sobre a Ferramenta

Página de Informações

A linguagem da Ferramenta de Programação Linear (ProLin) foi projetada de forma a ser tão próxima quanto possível da notação matemática de escrever modelos de Programação Linear. É de fácil uso. O botão Gerar Exemplo mostra exemplos aleatórios de como escrever programas de entrada, além de mostrar dicas adicionais sobre a linguagem.

Este sistema foi desenvolvido para apoiar o aprendizado de alunos e facilitar o ensino dos professores; não obstante, qualquer um pode usá-lo. Ele continua em desenvolvimento e ainda será aprimorado.

Aos interessados em aprender linguagens mais difundidas e com um maior poder de descrever modelos de entrada grandes e complexos, com o intuito da resolução eficiente dos mesmos, o autor sugere o estudo da linguagem GMPL (GNU Mathematical Programming Language, também conhecida como GNU MathProg Language). Ela é encontrada no GLPK (GNU Linear Programming Kit), nesta página.

O módulo resolvedor do sistema foi desenvolvido usando o API do GLPK para a linguagem C, versão 4.45.
