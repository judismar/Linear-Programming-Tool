Manual de Uso do Interpretador ProLinI

Atualmente disponível para Linux, é necessário fazer download do interpretador ProLinI, clicando aqui. A forma geral de usar o programa pelo Bash é:

./prolini-ptbr-32 arquivo_entrada -opção

O arquivo de entrada (de nome arquivo_entrada) contém o modelo de entrada em ProLin; em seguida, deve-se usar uma das opções para informar como o modelo será resolvido: -lin como programa linear; -int como programa inteiro; e -bin como programa binário.

A saída pode ser: (1) um aviso instrutivo de erro de entrada do usuário (e.g., esqueceu de incluir opção); (2) um pequeno relatório de erro de modelagem; (3) o resultado do programa. Ela é escrita no standard output do Linux.
