Manual de Uso da Ferramenta de Programação Linear

O componente gráfico principal da ferramenta é composto de:

  1. opção de escolha do tipo de problema;
  2. um editor de texto de entrada;
  3. uma caixa de texto de saída;
  4. Os botões Resolver, Gerar Exemplo e Limpar Resultado

1. Escolhendo o Tipo de Problema

Basta clicar na opção desejada: programa linear, inteiro ou binário. O modelo será resolvido de acordo com o tipo.

2. Editor de Texto de Entrada

O modelo é escrito ou copiado para o editor de texto. Ele possui tabulação, numeração e coloração dos elementos da linguagem ProLin.

3. Caixa de Texto de Saída

É apenas uma espécie de terminal de saída, onde são escrito o resultado do programa ou as dicas (ver 4).

O usuário não pode escrever nela.

4. Os Botões

  • Resolver: resolve o problema e imprime o resultado na caixa de texto de saída.
  • Gerar Exemplo: Gera um exemplo aleatório no editor de texto e dicas de uso do sistema na caixa de saída.
  • Limpar Resultado: Limpa a caixa de texto de saída.
